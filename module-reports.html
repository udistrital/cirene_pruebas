(inner) addListCriteria(report, spagoBIresponse)

Agrega los criterios o campos para analizar, especificados en reportes.json en el objeto reportes[].listCriteria, por ejemplo, hacer el PIE con Todos los distintos valores del "Tipo de Espacio". Estos criterios se agregan a un html select para ser seleccionados y automáticamente generar una gráfica.
Parameters:
Name Type Description
report object De tipo reportes[] que viene de reportes.json, con este debería ser el objeto del reporte actual.
spagoBIresponse object La respuesta de la petición ajax a la URL al servicio REST del DataSet asociado al reporte en SpagoBI.
